# Cypress-Test-SauceDemo

This project contains automated end-to-end tests developed with Cypress for the SauceDemo web application.

## Project Overview

The purpose of this project is to practice and demonstrate test automation skills using Cypress and JavaScript. The tests cover core user flows and functionality within the SauceDemo application.

## Technologies Used

- Cypress
- JavaScript
- Node.js

## Test Scope

The automated tests may include scenarios such as:

- User login
- Product verification
- Shopping cart functionality
- Checkout process
- Validation of expected application behavior

## Project Structure

- `Testare-Saucedemo.cy.js` – contains the automated test scenarios.
